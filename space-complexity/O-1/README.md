# Space Complexity O(1)

Write a function which follow the space complexity(O(1))

### Solution

<details>
<summary>Click Here for solution 1</summary>

```
function add(n1, n2) {
    return n1 + n2;
}
```

</details>

<details>
<summary>Click Here for solution 2</summary>

```
function multiply(n1, n2) {
    return n1 * n2;
}
```
